# 设置自然搜索归因

Source URL: https://support.appsflyer.com/hc/zh-cn/articles/15123194526353
Section: 归因概念
Updated: 2026-04-30T12:34:40Z

概要： 本文说明了如何针对已安装相关应用的用户，获取其再互动的自然搜索归因数据，帮助您优化自然搜索转化。

## 概览

自然搜索归因（Organic search attribution）是一种新型的归因解决方案。使用AppsFlyer的广告主可以通过该方案将再互动（即已激活相关应用的用户再次打开应用）归因到各种自然搜索引擎。

这部分独特的数据能为SEO（搜索引擎优化）成效提供重要的反馈信息，因此非常有助于SEO团队的工作。

#### 链路

- 用户在其移动端浏览器的搜索引擎（Google、Bing等等）中搜索某个关键词后，该搜索引擎显示了数条可点击的搜索结果。

- 您的网页开发人员对部分搜索结果进行了映射配置，使其通过安卓的App Link或iOS的Universal Link吊起相关应用。如果用户已安装相关应用，则点击这些搜索结果时就会直接吊起应用（不会先跳转到应用商店）。

- 应用打开时，AppsFlyer会收集必要的信息，并能够将此次应用打开归因到该用户使用的搜索引擎。

- 对于未安装应用的用户，您可以使用AppsFlyer的网页到应用解决方案（包括智能横幅、智能脚本和二维码）将这些用户引导至落地页，为其实现跳转并进行归因。

## 设置方式

前期准备 ：

- AppsFlyer Android SDK V6.12.1

- AppsFlyer iOS SDK V6.12.1

请按以下方式设置自然搜索归因 ：

- 请让您的开发人员根据相应的 安卓App Link 和 iOS Universal Link 指南（使用相关应用的顶级域名）实现应用吊起。

- 从AppsFlyer后台的侧边栏中选择 配置 > 应用配置 。然后启用：
- 自然搜索（SEO）移动应用归因 。

- 再互动归因 的开关。

#### 开发人员的注意事项

调用 performOnDeepLinking 会强制执行并解析深度链接URL，这会使SEO归因功能无法正常生效，导致相关转化无法归因到正确的SEO媒体渠道 。

## 数据可用性

数据可用性如下：

- 在面板和报告的汇总数据中，媒体渠道（media source）字段会显示带来相关用户的搜索引擎，比如： google_organic_seo 。

- 原始数据报告中：
- media source 字段会显示带来相关用户的搜索引擎，比如： google_organic_seo 。

- http_referrer 字段中包括搜索结果页面的完整URL，用户从该页面进入应用或跳转到移动端落地页。

- channel 字段的值为 seo 。

## 其他相关信息

### 使用搜索引擎列表

自然搜索归因支持下列搜索引擎：

- Google

- Bing

- Yahoo

- Ask

- DuckDuckGo

- Yandex

- Naver

- Daum

- 百度

- Ecosia

- Coccoc

- 搜狗

- Yahoo Japan

请注意 ：

- 如需对其他搜索引擎进行归因，请使用 旧版自然搜索归因解决方案 （需要开发人员介入）。

- 面板和报告以<search engine>_organic_seo的格式显示搜索引擎，例如： google_organic_seo 。

### 常见问题解答

问：使用自然SEO归因需要额外付费吗？

答：不需要，该功能已包含在您的套餐中。

问：自然SEO转化会计入我的套餐使用量吗？

答：是的，这类转化在套餐使用量中的统计方式与非自然转化相同。

问：SEO媒体渠道的名称可以自行调整吗？

答：不可以，自然SEO媒体渠道的名称是由AppsFlyer设定的，目前尚不支持自定义。

问：如果相关搜索引擎在自然搜索结果旁边展示安装按钮，用户点击该按钮后进入应用商店，AppsFlyer能将后续激活归因到这个搜索引擎吗？

答：在上述场景中，AppsFlyer无法收集到必要的归因信息。自然搜索归因仅适用于已安装相关应用的用户。

问：关键词数据是否可用？

答：关键词数据无法通过自然搜索归因解决方案获取，但在极少数情况中可能会有可用的安卓关键词数据。 详情请见此处说明。

问： 激活链路中支持自然搜索归因吗 ？

答：是的，对于已安装您应用的用户，您可以了解他们从哪个页面进入了您的网站。智能脚本可以收集这个页面的URL域名（也称为Web Referrer），然后在生成的链接中将其映射为指定参数的值。

Web Referrer尤其适用于识别引导用户到您网站的搜索引擎。面板和原始数据报告会在您指定的AppsFlyer参数下呈现该信息。

如需进一步了解通过智能脚本收集Web Referrer的方式，请参考 此文档 。

### 特点与局限性

特点 说明

浏览器
下列浏览器不支持自然搜索归因：

- 安卓中的FireFox

- iOS中除了Safari之外的所有浏览器
